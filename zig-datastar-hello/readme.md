## Zig Datastar :: Hello

This is an initial tryout of using [Datastart SDK for Zig 0.16](https://github.com/zigster64/datastar.zig). That SDK was minimally [patched](https://github.com/zigster64/datastar.zig/compare/main...dxps:datastar.zig:main) to work with Zig 0.16, and it's vendored in this repo.

Use the classic `zig build run` to run it.
